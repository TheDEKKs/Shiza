# Shiza 

This is telegram bot with AI for chatting in groups. He answers in aggressive mod. 

## Install
```bash
git clone https://github.com/TheDEKKs/Shiza
cd Shiza 
go mod tidy
```

Create .env
```.env
BOT_TOKEN=
TOKEN_API=
```

Token API you can get on openrouter.ai
And Bot Token you can get on the @BotFather


You can edit promt in the file: 
```bash
nvim ./internal/api/test.json 
```
