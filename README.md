# Shiza

This is telegram bot with AI for chatting in groups. He chatting in aggressive mod. But you can change promt and the very idea of the bot.

## Install
```bash
git clone https://github.com/TheDEKKs/Shiza
cd Shiza
go mod tidy
```

Create this is .env file in the root of the project
```.env
BOT_TOKEN=
TOKEN_API=
```

Token API you can get on openrouter.ai and you can use any model that they have.
And Bot Token you can get on the @BotFather


You can edit promt in the file:
```bash
nvim ./internal/api/promt.json
```
